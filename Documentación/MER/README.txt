MER — Modelo Entidad-Relación
==============================

Diagramas del modelo de datos (entidades, relaciones, cardinalidades).

Nota: El backend utiliza MongoDB Atlas (NoSQL). Los MER pueden representar el modelo conceptual de dominio y colecciones, o vistas lógicas equivalentes.

Formatos sugeridos: PNG, SVG, PDF, Draw.io, ERD en herramientas con export estándar.
