Librerías
=========

Dependencias y archivos necesarios para el funcionamiento del software.

Contenido típico:
- Listado de dependencias principales (backend: package.json; frontend: package.json / lockfile)
- Notas de versiones de Node.js, motor de base de datos, herramientas de build
- Licencias relevantes si el entregable lo exige

No subir node_modules completo al repositorio de documentación salvo indicación explícita del curso; preferir lockfiles y documentación.
