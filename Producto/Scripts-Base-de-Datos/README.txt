Scripts de base de datos
==========================

Archivos para creación de esquemas, índices, seeds o migraciones.

PetConnect utiliza MongoDB Atlas (NoSQL). Los archivos aquí pueden ser:
- Scripts mongosh (.js)
- Agregaciones o exportaciones JSON de ejemplo
- Documentación de índices y colecciones
- Enlaces a migraciones del repositorio backend

Si el programa académico pide SQL, incluye una nota explicando la equivalencia conceptual con MongoDB o un diagrama lógico en Documentación/MER.
